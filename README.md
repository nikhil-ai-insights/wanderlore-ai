## ✨ Overview

Wanderlore AI transforms ordinary travel into meaningful cultural experiences through Generative AI. Instead of recommending only popular attractions, it uncovers hidden gems, local traditions, immersive storytelling, authentic food, cultural etiquette, festivals, and personalized itineraries tailored to each traveler.

---

## 🚀 Key Features

- 🌍 AI-Powered Destination Discovery
- 🗺️ Hidden Gems Recommendation
- 📖 Cultural Storytelling & Heritage Explorer
- 🎉 Local Events & Festival Discovery
- 🍜 Authentic Local Food Recommendations
- 📅 Personalized AI Travel Itineraries
- ❤️ Save & Manage Trips
- 📄 Export Travel Guides (PDF)
- 🔐 Secure Firebase Authentication
- ☁️ Cloud Firestore Integration
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS
- **Backend:** Firebase Authentication, Cloud Firestore
- **AI:** Google Gemini API
- **Animations:** Framer Motion
- **Deployment:** Firebase Hosting / Vercel

---

## ⚡ How It Works

1. Sign up or log in securely.
2. Choose your destination and travel preferences.
3. AI analyzes your interests.
4. Generate personalized cultural travel recommendations.
5. Save, revisit, and export your travel guides.

## 🎯 Why Wanderlore AI?

Unlike traditional travel platforms, Wanderlore AI focuses on **culture, authenticity, and storytelling** rather than just attractions. It helps travelers connect with places through history, local traditions, hidden experiences, and AI-powered insights.
