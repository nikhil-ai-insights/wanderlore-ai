# 🤝 Contributing to Wanderlore AI

Thank you for your interest in contributing to **Wanderlore AI**! 🚀

We welcome developers, designers, AI enthusiasts, and open-source contributors to help make cultural travel more immersive and accessible through AI.

## 🌟 Ways to Contribute

You can contribute by:

- 🐛 Fixing bugs
- ✨ Building new features
- 🎨 Improving UI/UX
- ⚡ Optimizing performance
- 🔒 Enhancing security
- 📱 Improving responsiveness
- ♿ Improving accessibility
- 📖 Improving documentation
- 🧪 Writing tests

---

## 🚀 Getting Started

### 1. Fork the repository

Click the **Fork** button at the top right of this repository.

### 2. Clone your fork

```bash
git clone https://github.com/your-username/wanderlore-ai.git
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

---

## 🌿 Create a Branch

Always create a new branch before making changes.

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b feature/add-ai-storytelling
```

---

## 💻 Coding Guidelines

Please ensure that your contribution:

- Follows the existing project architecture
- Uses reusable components
- Keeps code clean and readable
- Avoids duplicate code
- Includes meaningful commit messages
- Maintains responsiveness
- Preserves accessibility standards

---

## ✅ Before Opening a Pull Request

Please verify that:

- The project builds successfully
- No console errors exist
- All features work correctly
- Responsive design is maintained
- Accessibility is not affected
- Existing functionality remains unchanged

---

## 📝 Commit Message Format

Examples:

```text
feat: add hidden gems module

fix: resolve Firebase authentication issue

refactor: optimize AI recommendation service

docs: update README

style: improve responsive layout
```

---

## 🚀 Pull Requests

When submitting a Pull Request, include:

- A clear description of your changes
- The reason for the changes
- Screenshots (if UI was modified)
- Testing performed

---

## 🐞 Reporting Issues

If you find a bug, please include:

- Operating System
- Browser
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

---

## ❤️ Thank You

Every contribution helps improve **Wanderlore AI** and makes AI-powered cultural travel more meaningful for everyone.

Happy Coding! 🚀
